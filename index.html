WoW Class Picker (Web Application)

A simple front-end web app to randomly select a playable class for the video game World of Warcraft, given a user's preferences as parameters.

Languages/Skills/etc: HTML, CSS, JS, jQuery

Do Your Chores (iOS Application)

A mobile application designed to randomly designate chore assignments. A user must input the names of each person they wish to have chores assigned to, as well as each chore to be completed. The app will then assign an even amount of unique chores randomly to each person on the user's list. If the user is satisfied, the session can be saved and referenced at their convenience.

Languages/Skills/etc: Swift 4, CoreData, Web APIs

Game of Zones (Web Application)

Game of Zones is a web application where teams of players compete against each other to eliminate excess energy usage. Teams earn more points for using less energy. Currently, three energy types are tracked: In-home Gas, Electricity, and MPG. The app is embedded in Facebook, allowing us to utilize their PHP Graph SDK to retrieve user data to structure our database.

Languages/Skills/etc: HTML, CSS, JS, PHP, MySQL
